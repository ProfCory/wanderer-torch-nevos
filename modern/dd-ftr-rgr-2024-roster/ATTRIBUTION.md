# Attribution

This project is a lightweight character tracker and does not reproduce large sections of rule text.

Some game rule structure (e.g., lists of feature names by level) is inspired by publicly available 2024 rules references.

Rules reference links in the UI point to:
- D&D Beyond Basic Rules (2024): https://www.dndbeyond.com/sources/dnd/br-2024
- Roll20 Free Basic Rules (2024): https://roll20.net/compendium/dnd5e/Free%20Basic%20Rules%20%282024%29

If you later add SRD 5.2/5.2.1 text or data into this repository, ensure you comply with CC-BY 4.0 attribution requirements for that SRD content.
